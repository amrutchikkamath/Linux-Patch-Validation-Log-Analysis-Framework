# Linux Patch Validation & Log Analysis Framework

## Overview
This project simulates enterprise-level patch validation testing in a Linux environment.

## Features
- Simulated patch installation using Bash
- Log analysis using Python
- Automated validation report generation
- Basic regression & failure detection logic

## How to Run

### 1. Apply Patch
cd scripts
bash apply_patch.sh

### 2. Run Log Analyzer
python3 log_analyzer.py

### 3. Check Report
Open the 'reports' folder to view validation_report.txt

## Technologies Used
- Linux (Bash)
- Python
- Log Parsing
- Simulated Patch Testing
